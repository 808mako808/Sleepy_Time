Sleepy Time

Privacy Policy

Effective date: April 9, 2026

Let's Sleep is a simple Alexa skill that plays a sleep audio track on supported Alexa devices.

  • Let's Sleep stores the Dropbox audio link that a user provides on the setup page.
  • Let's Sleep stores that link only so the Alexa skill can play the requested audio track.
  • Let's Sleep does not use account linking.
  • Let's Sleep does not process payments or purchases.
  • Let's Sleep does not contain advertising.
  • Let's Sleep does not sell user information.

This page is provided to explain how Let's Sleep handles user data for Alexa certification and publication.
